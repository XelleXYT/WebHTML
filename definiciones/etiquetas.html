Etiquetas


Definición:

Una etiqueta es, dentro de la sintaxis del HTML, una forma de abarcar un objeto (Un texto, una imágen, una tabla, etc.) y darle formato para que pueda ser representado correctamente en un navegador.


Usos:

Las etiquetas siempre han de ir en conjunto, si una abre otra ha de cerrar. El cirre de una etiqueta siempre es la misma con el caracter "/" añadido entre "<" y el nombre de la etiqueta.

Ejemplo: <p>Esto es un párrafo.</p> - Comienza con una etiqueta <p> y finaliza con una etiqueta </p>


Etiquetas únicas:

Esta lista reune las etiquetas que solo aparecerón dos veces (una abriendo y otra cerrando) en un documento HTML:

  • Etiqueta <html>: Es la etiqueta utilizada para que el navegador reconozca que el contenido de esta es código HTML.

  • Etiqueta <head>: Es la etiqueta utilizada para determinar el contenido del título de la pestaña o ventana del navegador.

  • Etiqueta <body>: Es la etiqueta utilizada para determinar el contenido de la ventana del navegador.
