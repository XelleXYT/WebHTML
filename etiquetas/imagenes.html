Imágenes

Formato:

Para introducir una imagen se utiliza la etiqueta:

  • <img para el inicio y /> para el final.

Atributos:

La etiqueta img dispone de varios atributos:

  • align - valor - Para alinear la imagen
    • top
    • bottom
    • middle
    • left
    • right

  • alt - texto - Texto alternativo

  • border - pixels - Tamaño del borde

  • crossorigin - valor - Acceso a webs de terceros
    • anonymous
    • use-credentials

  • height - pixels - Alto de la imagen

  • hspace - pixels - Margen con el texto horizontal

  • ismap - ismap - Mapa de la imagen

  • longdesc - URL - Descripción detallada de la imagen

  • sizes - Tamaño de la imagen

  • src - URL - Ruta de la imagen

  • srcset - URL - Ruta de la imagen variante

  • usemap - #nombremapa - Usar mapa de la imagen

  • vspace - pixels - Margen con el texto vertical

  • width - pixels - Ancho de la imagen
