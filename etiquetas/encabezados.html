Encabezados

Formato:

Dependiendo el tipo de encabezado que queramos utilizar usaremos una u otra etiqueta. Las distintas etiquetas de encabezado son:

  • <h1>: Para los encabezados más grandes.
  • <h2>: Para los encabezados menos grandes.
  • <h3>: Para los encabezados intermedios.
  • <h4>: Para los encabezados del mismo tamaño que el texto.
  • <h5>: Para los encabezados menos pequeños.
  • <h6>: Para los encabezados más pequeños.

Atributos:

Las etiquetas de encabezado disponen de 1 solo atributo.

Este atributo es align, que sirve para alinear el encabezado y puede ser utilizado con los valores:

  • left: Alinea el encabezado a la izquierda.
  • center: Alinea el encabezado al centro.
  • right: Alinea el encabezado a la derecha.
  • justify: Alinea el texto de forma justificada.
