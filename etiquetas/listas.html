Listas

Formato:

Hay varios tipos de listas, estos se muestran a continuación:

  • <ul> para el inicio y </ul> para el final de una lista desordenada.
  • <ol> para el inicio y </ol> para el final de una lista ordenada.
  • <dl> para el inicio y </dl> para el final de una lista de definiciones.

A su vez estas disponen de <li></li>, <dt></dt> para las de definiciones, que son utilizados para asignar valores en la lista.


Atributos:

Las etiquetas disponen de distintos atributos:

  • Las listas desordenadas:

    • compact - compact - Especifica si la lista debe ser compactada

    • type - Especifica el símbolo a utilizar al comienzo del valor
      • disc
      • square
      • circle

  • Las listas ordenadas:

    • compact - compact - Especifica si la lista debe ser compactada

    • reversed - reversed - Especifica si el orden de los valores de la lista está invertido

    • number - número - Especifica el valor por el que empieza la lista

    • type - Especifica el tipo de caracter a utilizar al comienzo del valor
      1. 1
      2. A
      3. a
      4. I
      5. i

  • Las listas de definiciones solo disponen de definiciones introducidas en:<dt></dt>

  • Valores de las listas (<dt></dt>):

    • type - Lo heredan de <ul> u <ol> y les sirve para cambiar su valor propio
    • value - Solo funciona con las <ol> y es heredado
