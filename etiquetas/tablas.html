Tablas

Formato:

Para crear una tabla hacen falta varias etiquetas:

  • <table> para el inicio y </table> para el final de la tabla.
  • <tr> para el inicio y </tr> para el final de las filas.
  • <td> para el inicio y </td> para el final de las celdas.
  • <th> para el inicio y </th> para el final de las celdas-encabezado.

Para crear una tabla siempre se ha de seguir la estructura de:

  1. Creamos la tabla con <table>
  2. Creamos las filas con <tr>
  3. Creamos las celdas dentro de las filas con <td>
  4. Los <td> que queramos como encabezados los sustituiremos por <th>
