Formatos

Definición:

Los formatos se utilizan para resaltar o marcar el texto. Hay varios tipos de formatos, estos pueden ser aplicados unos sobre otros.

Las etiquetas de los formatos son:

  • <b>: Para el texto en negrita.
  • <strong>: Para el texto importante.
  • <i>: Para el texto en itálica.
  • <em>: Para el texto enfatizado.
  • <mark>: Para el texto marcado.
  • <small>: Para el texto pequeño.
  • <del>: Para el texto borrado.
  • <ins>: Para el texto insertado.
  • <u>: Para el texto subrayado.
  • <sup>: Para los supraíndices.
  • <sub>: Para los subíndices.

Atributos:

Las etiquetas de formato de texto no disponen de atributos, excepto de los globales.
