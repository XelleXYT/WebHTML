Enlaces

Formato:

Para introducir una imagen se utiliza la etiqueta:

  • <a> para el inicio y </a> para el final.

Atributos:

La etiqueta a dispone de varios atributos:

  • charset - codificación - Especifica la codificación del documento enlazado

  • coords - coordenadas - Especifica las coordenadas del enlace

  • download - archivo - Especifica el archivo que será descargado

  • href - URL - Especifica la dirección a la que será dirigido

  • hreflang - codigo_idioma - Especifica el idioma del documento

  • media - query - Especifica el formato del archivo

  • name - name - Especifica el nombre del ancla

  • rev - text - Especifica la relación entre el documento actual y el enlazado
